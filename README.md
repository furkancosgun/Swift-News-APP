
Swift News App
- 
News app built in Swift 
## Features

-   Most popular news
-	Online data
-	Redirecting to news site via Safari
-	For personal development

## Tech Stack

Developed using  swift with xcode. 
API: <a href="https://collectapi.com/tr/api/news/haberler-api">Collect Api</a>

## Development Setup

Clone the repository and run the following commands:
```
open with xcode this project
run it!
```
